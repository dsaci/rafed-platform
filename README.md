# رافد — المرافقة الرقمية التربوية

> **نرصد، نرافق، نحسّن**

منصة رافد هي المرافق الرقمي لمنظومة التربية الوطنية الجزائرية. تهدف المنصة إلى رصد المشاكل التقنية التي يواجهها مستخدمو المنصات الرقمية التربوية وتقديم حلول فورية باستخدام الذكاء الاصطناعي.

---

## 🏗️ التقنيات المستخدمة

| التقنية | الوصف |
|---------|-------|
| Next.js 14 | إطار العمل (App Router) |
| TypeScript | لغة البرمجة |
| Tailwind CSS | التصميم |
| Framer Motion | الحركات والانتقالات |
| Supabase | قاعدة البيانات والمصادقة |
| Anthropic Claude | الذكاء الاصطناعي |
| Vercel | الاستضافة |

---

## 📦 التثبيت

```bash
# استنساخ المشروع
git clone https://github.com/your-username/rafed.git
cd rafed

# تثبيت الاعتماديات
npm install

# نسخ ملف المتغيرات البيئية
cp .env.example .env.local

# تعديل المتغيرات البيئية
# NEXT_PUBLIC_SUPABASE_URL=
# NEXT_PUBLIC_SUPABASE_ANON_KEY=
# SUPABASE_SERVICE_ROLE_KEY=
# ANTHROPIC_API_KEY=

# تشغيل خادم التطوير
npm run dev
```

---

## 🗄️ قاعدة البيانات

قم بتنفيذ ملفات الترحيل في Supabase:

1. `supabase/migrations/001_initial_schema.sql` — الجداول والسياسات والبيانات الأولية
2. `supabase/migrations/002_auth_trigger.sql` — إنشاء ملف تعريف تلقائي عند التسجيل

## 📁 هيكل المشروع

```
src/

│   ├── issues/            # إرسال وتتبع الانشغالات

│   ├── dashboard/         # لوحة تحكم المشرف
│   ├── login/             # تسجيل الدخول
│   └── api/               # واجهات برمجية
│       ├── diagnose/      # التشخيص الذكي
│       └── reports/       # توليد التقارير
├── components/
│   ├── ui/                # مكونات واجهة المستخدم
│   ├── layout/            # مكونات التخطيط
│   └── forms/             # نماذج الإدخال
├── lib/                   # المكتبات والأدوات
│   ├── supabase.ts       # عميل Supabase
│   ├── constants.ts      # الثوابت والبيانات

│   └── validators.ts    # مخططات التحقق
└── types/                 # أنواع TypeScript
    └── index.ts
```

## ⚙️ المتغيرات البيئية

| المتغير | الوصف |
|---------|-------|
| `NEXT_PUBLIC_SUPABASE_URL` | رابط مشروع Supabase |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | مفتاح Supabase العام |
| `SUPABASE_SERVICE_ROLE_KEY` | مفتاح الخدمة (سري) |
| `ANTHROPIC_API_KEY` | مفتاح Anthropic Claude |
| `NEXT_PUBLIC_APP_URL` | رابط التطبيق |
| `NEXT_PUBLIC_APP_NAME` | اسم التطبيق |

## 📄 الصفحات

| الصفحة | المسار | الوصف |
|--------|--------|-------|
| الرئيسية | `/` | عرض المنصات والإحصائيات |
| المنصات | `/platforms` | تصفح وفلترة المنصات |
| تفاصيل المنصة | `/platforms/[id]` | شروحات وانشغالات المنصة |
| إرسال انشغال | `/issues/new` | نموذج متعدد الخطوات |
| تتبع الانشغال | `/issues/[id]` | حالة التذكرة والتشخيص |
| مكتبة المحتوى | `/content` | فيديوهات ومقالات |
| تفاصيل المحتوى | `/content/[slug]` | عرض المحتوى |
| تسجيل الدخول | `/login` | للمشرفين فقط |
| لوحة التحكم | `/dashboard` | إدارة الانشغالات والمحتوى |

## 🤖 الذكاء الاصطناعي

- تحديد مستوى الثقة في الحل

# المساهمة في رافد

نرحب بمساهماتكم في تطوير منصة رافد. يرجى اتباع الإرشادات التالية:

## 🌿 استراتيجية الفروع

- `main` — الإنتاج
- `dev` — التطوير
- `feature/*` — الميزات الجديدة

## 📝 رسائل الإيداع

نستخدم صيغة Conventional Commits باللغة العربية:

```
feat: إضافة نموذج الانشغالات
fix: إصلاح مشكلة المصادقة
chore: تحديث المتغيرات البيئية
docs: تحديث ملف القراءة
style: تحسين تصميم البطاقات
refactor: إعادة هيكلة مكون المنصات
```

## 🔄 سير العمل

1. أنشئ فرعاً من `dev`
2. طوّر الميزة واختبرها محلياً
3. أرسل طلب سحب (Pull Request) إلى `dev`
4. انتظر مراجعة الكود

## 🧪 قبل الإرسال

- تأكد من عدم وجود أخطاء TypeScript: `npx tsc --noEmit`
- تأكد من نجاح البناء: `npm run build`
- تأكد من عدم وجود `console.log` في الكود
